Django Components Bootstrap

Bootstrap 5 components for Django, inspired by the React-Bootstrap API.

Installation

pip install django-components-bootstrap

Quick Start

Add to your INSTALLED_APPS:

INSTALLED_APPS = [
    # ...
    "django_components",
    "django_components_bootstrap",
]
